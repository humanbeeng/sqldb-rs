# sqldb-rs

Memory backed SQL database. 

Rust implementation of [blog](https://notes.eatonphil.com/database-basics.html) in order to learn Rust, lexers and parsers. Not meant for actual use. 

Supports.
- SELECT
- INSERT
- CREATE
